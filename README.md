# DarkflameShadow

A simple RPG game made with Java base on OOP concept. Fight monsters, get stronger, and collect cool gear!

## What is this game?

This is my first RPG game project. You play as a character who fights monsters and bosses. When you win, you get money and experience points. Use your money to buy new equipment or upgrade your stats.

## How to play

1. **Start the game** - Run the JAR file or compile the Java code
2. **Look at your stats** - You can see your level, attack power, HP, and money
3. **Go travel** - Find monsters to fight
4. **Fight and win** - Your character attacks automatically
5. **Get rewards** - Earn money and EXP when you win
6. **Get stronger** - Level up and buy better gear
7. **Keep playing** - Enemies get harder as you progress

## Game features

- **Fighting system** - Your character fights monsters and bosses
- **Level up** - Get EXP to become stronger
- **Equipment** - Roll for random gear (weapon, helmet, armor, pants, boots)
- **Money** - Buy upgrades with gold from battles

## How to run

You need Java installed on your computer.

**Easy way:**
```
java -jar RngRpgGame.jar
```

**Or compile it yourself:**
```
javac RunGame.java
java RunGame
```
